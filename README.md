# 数独游戏

一个精美的网页版数独游戏，采用清淡精致的配色设计。

## 功能特性

- **9x9 标准数独格式**
- **5种难度等级**：简单、中等、困难、极难、自定义
- **自定义模式**：手动填写初始数字，然后开始游戏
- **自动校验**：实时检测错误并记录错误次数
- **笔记模式**：在格子中标记候选数字
- **提示功能**：随机填入一个正确答案
- **计时器**：记录游戏用时
- **完成庆祝**：彩色纸屑动画庆祝胜利
- **图片保存**：将原始布局和完成布局保存为PNG图片

## 使用方法

1. 直接在浏览器中打开 `sudoku.html` 文件
2. 选择难度等级开始游戏
3. 点击格子选中，使用数字键盘或键盘输入数字
4. 完成后点击"保存图片"下载纪念图

## 操作说明

- **鼠标点击**：选择格子
- **数字键 1-9**：输入数字
- **Delete/Backspace**：清除数字
- **方向键**：移动选择

## 技术栈

- 纯 HTML + CSS + JavaScript
- 无需任何外部依赖
- 响应式设计，支持移动端
